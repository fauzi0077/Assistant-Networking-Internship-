### Firewall Aliases Configuration (pfSense)

This image shows the configuration of **Firewall Aliases** in pfSense, specifically for **port aliases**.  
Aliases are used to group multiple ports or services under a single name, making firewall rule management more efficient and easier to maintain.

In this configuration, commonly used service ports such as:
- Remote Desktop (3389)
- LDAP and LDAP over SSL
- Kerberos
- Skype and WhatsApp
- Desktop Central services

are grouped into aliases to allow centralized control.

Using aliases improves:
- Firewall rule readability
- Faster configuration changes
- Easier maintenance of security policies

This method is commonly applied in enterprise networks to simplify access control and port management.
